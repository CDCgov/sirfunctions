# Helper function to generate stool adequacy on a rolling basis

Helper function to generate stool adequacy on a rolling basis

## Usage

``` r
stool_ad_rolling(stool.data, pop.data, start_date, end_date, spatial_scale)
```

## Arguments

- stool.data:

  AFP dataset containing the adequacy final column

- pop.data:

  population data

- start_date:

  start date

- end_date:

  end date

- spatial_scale:

  "ctry", "prov", or "dist".

## Value

summary table containing stool adequacy on a rolling basis
