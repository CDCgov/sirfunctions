# Fetch the Rds file to be used for a desk review

Fetch the Rds file to be used for a desk review

## Usage

``` r
fetch_dr_data(country, year, local_dr_repo)
```

## Arguments

- country:

  country name as a string

- year:

  year as an integer

- local_dr_repo:

  the local desk review repository

## Value

A status message
