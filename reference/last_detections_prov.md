# Identify last detections at the province level

Identify last detections at the province level

## Usage

``` r
last_detections_prov(data_p, .start_date, .end_date)
```

## Arguments

- data_p:

  dataset containing epids with cases

- .start_date:

  `date` Start date.

- .end_date:

  `date` End date.

## Value

`tibble` Summary table identifying last detections at the province level
