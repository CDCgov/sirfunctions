# Create the KPI template

Create the KPI template

## Usage

``` r
generate_kpi_template(output_path, name, edav)
```

## Arguments

- output_path:

  `str` Where to output the script to.

- name:

  `str` Name of the KPI folder.

- edav:

  `logical` Whether the init pulled data from EDAV or not.

## Value

None.

## Examples

``` r
if (FALSE) { # \dontrun{
generate_kpi_template(getwd(), "test_folder")
} # }
```
