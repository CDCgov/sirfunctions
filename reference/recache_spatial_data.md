# Should the local spatial cache be updated?

Should the local spatial cache be updated?

## Usage

``` r
recache_spatial_data(analytic_folder, spatial_folder, edav, output_format)
```

## Arguments

- analytic_folder:

  `str` Path to the analytics folder.

- spatial_folder:

  `str` Path to the spatial folder.

- edav:

  `logical` Use EDAV?

- output_format:

  `str` Output format the spatial files are in.

## Value

`logical` Should we recache the spatial data or not?
