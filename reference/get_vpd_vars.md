# Obtain available variables for VPD data

`lifecycle::badge("experimental")`

Obtains the variable names available for each vaccine preventable
disease (VPD) in the `variable` column.

## Usage

``` r
get_vpd_vars()
```

## Value

`tibble` A tibble with the available variable choices.

## Examples

``` r
if (FALSE) { # \dontrun{
vpd_vars <- get_vpd_vars()
} # }
```
