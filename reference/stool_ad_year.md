# Helper function to generate stool adequacy on a non-rolling basis

Helper function to generate stool adequacy on a non-rolling basis

## Usage

``` r
stool_ad_year(stool.data, pop.data, year.data, spatial_scale)
```

## Arguments

- stool.data:

  AFP data with adequacy final column

- pop.data:

  population data

- year.data:

  table generated by the generate_year_data() function

- spatial_scale:

  "ctry", "prov", or "dist

## Value

summary table of stool adequacy on a yearly basis
